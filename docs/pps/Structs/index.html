    Struct vs Union

    Material de apoyo

    Enlace de descarga aquí. El ejercicio presenta el código de un programa con las siguientes características:

    • en el archivo cabecera Libros.h se declara un tipo Libro que es un struct (Libros)
    • se procesan los argumentos en la línea de orden para interactuar con el programa
    • se leen y escriben archivos en modo binario y modo texto
    • en particular, muestra en pantalla (salida estándar) los contenidos de un fichero que se le pasa como argumento

    Sintaxis de invocación del programa

    • ./LibrosIO [1|2|3]
    • ./LibrosIO 4 nombre_archivo
